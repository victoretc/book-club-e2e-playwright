# e2e node playwright тесты для Читальной 

e2e тесты для приложения [Читальной](https://github.com/victoretc/book-club)
Allure отчеты доступны по ссылке: <https://victoretc.github.io/book-club-e2e-playwright/>

## Запуск

1. Запустить [Читальную](https://github.com/victoretc/book-club)
2. npm i 
3. npx playwright install --with-deps chromium
4. npm run e2e
